Web Development - Best Practices

The following guidelines are provided as a broad overview of web development best practices, suggestions and resources.

Development Best Practices

  • Don’t reinvent the wheel. Use styles, templates, code that have already been developed.
  • Search and use available guidelines like the WIKI.
  • Check spelling and grammar in text and code.
  • Check pages on all browsers, including older versions of IE.
  • Check console output for warnings or errors.
  • Use semantic code.
  • Abstract code and css from page.
  • Simplify as much as possible.
  • Compress and minimize code to decrease load times.
  • Design with mobile in mind.
  • Build in sufficient testing time.
  • Set time aside to write documentation.

HTML and CSS

  • Avoid inline styles.
  • Use CSS classes for styling and id for unique identifiers.
  • Don’t use !important in CSS. This overrides any cascading rules and can frustrate debugging.
  • Use semantic code.
  • Use tools such as Lint and Tidy to check for code anomalies.
  • Check for broken links.
  • Properly size images. Don’t use a 1000px wide image and then resize it to 100px.
  • If there are many icons or images combine them into a sprite.
  • Validate the html. There are a number of tools like Firebug that can do this as well as online services such as W3C HTML Validator.

Javascript

  • Place script loaders and javascript at bottom of pages.
  • Do not place javascript code among html. This blocks the page.
  • Make use of jQuery framework for writing code.
  • Avoid inline code for events (eg. <a onclick="doSomething()" href="#">Click!</a>)
  • Don’t always depend on javascript. What if javascript was turned off? Remember that this is what search engines see.
  • Run JSLint/JSHint to validate code.
  • Use dependencies to load scripts.
  • CamelCase or lowercase? Tabs or spaces? Hyphens or underscores? Picking a system and being consistent is more important than which you choose.
  • Don't leave console debugging messages in your production code.
